1: How to use editor

• Select parts of the Live UI and change their color

• Click “Download File” when done

• If you rename the file make sure “.ask” is still the file's extension

• Place the file in the apropriate directory

2: Directory

For Mac:
Find the Ableton 12 application in your Applications folder, right click on it and select Show Package Contents. Go to Content > App Resources > Themes and drop your theme there.

For Windows:
Drop the theme into C:\ProgramData\Ableton\Live 12 Suite\Resources\Themes.

3: Last Steps

• After placing your file in the correct folder,

• Run Live 12 and open the preferences window.

• Go to “Theme & Colors”

• Your new theme is in the “Theme” drop-down.
